# PRISM AI – GitHub Summary

## Short Description (for GitHub "About" / repo description)

**PRISM AI** – Conversational Business Intelligence for sales patterns. Ask natural-language questions, get pattern analysis, forecasts, and insights powered by Databricks, advanced signal processing (STFT, wavelets, HHT), and LLMs.

## One-Line Tagline

## Feature Summary

| Area | Features |
|------|----------|
| **Conversational BI** | Natural language questions about sales patterns, trends, spikes, volatility, seasonality |
| **Data Layer** | Databricks Unity Catalog, pattern metadata, daily sales, forecasts |
| **Forecasting** | Pattern-aware predictions with business narratives from LLM |
| **Transforms** | STFT, Wavelet (CWT), Hilbert–Huang Transform for pattern detection |
| **Stack** | Next.js 14, FastAPI, Redis cache, NVIDIA NIM (LLM) |

---

## Tech Stack

- **Frontend:** Next.js 14, React 18, Tailwind CSS, Lucide icons
- **Backend:** FastAPI, Pydantic
- **Data:** Databricks SQL + Unity Catalog
- **LLM:** NVIDIA NIM (Llama 3.1)
- **Cache:** Redis
